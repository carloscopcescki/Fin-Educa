# FinEduca 🏦

## Descrição
FinEduca se trata de um website que tem como objetivo, promover a cultura da educação financeira para os estudantes. Reunindo artigos, notícias, calculadoras e ferramentas para facilitar o aprendizado e tornar o tema mais acessível para os jovens.

## Ferramentas utilizadas
- HTML
- CSS
- JavaScript
